# 🛒 Cartify

Cartify is a modern Android shopping application developed using Java in Android Studio.

## ✨ Features
- User Login
- Home Screen
- Categories
- Product Listing
- Shopping Cart
- Wishlist
- Modern UI Design

## 🛠️ Tech Stack
- Java
- Android Studio
- RecyclerView
- Material Design
- Firebase
